# Using an AWS E3 VM as an agent to stream live data continuously into Foundry

Optimized VM SKU config for data streaming agent:
- SKU: t3 micro (2vCPUs, 1GiB RAM) - cheapest
- Security group: SSH inbound enabled, HTTP outbound enabled
- Disk: General purpose SSD 8GiB (3000 IOPS)

On the VM:
- update packages
```
sudo apt update
```
- install python libraries: websockets and requests
```
sudo apt install -y python3-websockets python3-requests
```

Establish Connection to Foundry:
- In Foundry: create stream data set, then follow instruction to generate token or secret, I used token
- create a python file on the VM and paste the in Foundry autogenerated script, to avoid errors, you might need to change the timestamp in the script to this: "timestamp": 1713207505000
```
nano app.py
```
- run test script
```
python3 app.py
```
